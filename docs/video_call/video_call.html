Peer-to-peer video calling using WebRTC

How to use:

  • Click "Start Call" to begin a new call session
  • Share the generated room ID or QR code with the person you want to call
  • They can join by entering the room ID and clicking "Join Call"
  • Allow camera and microphone permissions when prompted
  • For cross-device calls, you'll need to manually share signaling data
Note: This app uses manual signaling for cross-device communication. You'll need to copy and paste the signaling data between devices.
